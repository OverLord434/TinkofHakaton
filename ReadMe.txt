Инструкция по развертыванию Django проекта из архива
Шаг 1: Установка Python и pip

    Проверьте наличие Python:
    Откройте терминал (командную строку) и введите:

    	sh

	python --version

Если Python не установлен, скачайте его с официального сайта и установите.

Проверьте наличие pip:
Введите в терминале:

	sh

	pip --version

Если pip не установлен, установите его, скачав get-pip.py и запустив команду:

	sh
	
    	python get-pip.py

Шаг 2: Разархивирование проекта

    Разархивируйте проект:
        На Windows: Щелкните правой кнопкой мыши на архив и выберите "Извлечь все..."

Шаг 3: Создание виртуального окружения

    Перейдите в директорию проекта:

    	sh

	cd path/to/extract/project_folder

Создайте виртуальное окружение:

	sh

	python -m venv venv

Активируйте виртуальное окружение:

    	sh

	venv\Scripts\activate

Шаг 4: Установка зависимостей

    Убедитесь, что вы находитесь в активированном виртуальном окружении, и установите зависимости проекта:

    sh

    pip install -r requirements.txt

Шаг 5: Настройка проекта
	
Примените миграции базы данных:

	sh

	python manage.py makemigrations

	python manage.py migrate

Создайте суперпользователя для доступа к админке:

	sh

    	python manage.py createsuperuser

Шаг 6: Запуск сервера разработки

    Запустите сервер разработки:

    sh

python manage.py runserver

Откройте браузер и перейдите по адресу:

arduino

    http://127.0.0.1:8000
